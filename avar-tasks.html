How to set up the Power Automate flow:
1. Go to make.powerautomate.com
2. Click + Create → Automated cloud flow
3. Trigger: "When an HTTP request is received" → Method: POST
4. Paste this into "Request Body JSON Schema": 
{"type":"object","properties":{"to":{"type":"string"},"subject":{"type":"string"},"body":{"type":"string"},"event":{"type":"string"}}}
5. Add step: Send an email (V2) (Office 365 Outlook)
6. To = triggerBody()?['to']
7. Subject = triggerBody()?['subject']
8. Body = triggerBody()?['body']
9. Save, then copy the HTTP POST URL and paste it above
